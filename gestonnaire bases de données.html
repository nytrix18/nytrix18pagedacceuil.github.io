Le gestionnaire base de données est responsable du système de gestion, s'assure de la sécurité des données numériques de son entreprise et rend ces informations accessibles aux collaborateurs. Le gestionnaire base de données est également connu sous le nom d'administrateur de base de données

Le Gestionnaire Base de Données s'occupe principalement de créer, administrer, organiser et mettre à jour la base de données ; veille à ce que les informations soient disponibles à tout moment afin que les collaborateurs des autres serveurs puissent les utiliser en temps voulu ; fait évoluer la base de données en lui
